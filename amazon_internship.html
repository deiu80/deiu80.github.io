Technical skills

I gained experience in agile software development methodologies, which helped me to understand how to manage and own projects effectively and deliver high-quality software in a fast-paced environment.

Acquired a good understanding of serverless architecture, including distributed systems and the principles of RESTful API design.

Utilized a wide range of AWS services such as API Gateway, DynamoDB, Lambda, IAM, CloudWatch, CloudFormation

I also have learned about software testing and quality assurance, which are critical components of building robust and scalable software. I have conducted automated testing using TestNG framework, ensuring high-quality software delivery.

Deliverying my project required Java for building APIs and all serverless functionality in AWS while Angular and TypeScript were used for web development.

Career Development

Due to a highly collaborative work environment, and through working on a real-world project with my peers, I learned how to effectively communicate and share ideas with others.

The internship provided me with valuable insights into the tech industry and the culture at Amazon. I had the opportunity to work alongside some of the best software engineers and gain exposure to real-world challenges. Additionally, Amazon's strong focus on mentorship and career development helped me to build my professional network and get guidance on how to navigate the tech industry as a young professional.
